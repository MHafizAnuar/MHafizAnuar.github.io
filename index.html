Smart Home Automation Via IoT

Traditional electrical sockets require manual operation, which can be inconvenient and inefficient, especially when users need remote or automated control. The lack of integration with IoT systems limits energy management, increases energy wastage, and reduces convenience in modern smart homes. To address these issues, a home automation system using an arduino microcontroller was developed to enable remote control and monitoring of electrical sockets through IoT technology. This solution aims to enhance energy efficiency, convenience, and adaptability in home environments.

Plant Monitoring Via IoT

Modern gardening and agriculture face challenges in ensuring optimal plant health due to factors such as irregular watering, unpredictable weather conditions, and lack of real-time monitoring. These issues can lead to inefficient use of resources like water, as well as poor plant growth or yield.
To address these problems, a plant monitoring system was developed using an ESP32 microcontroller, equipped with sensors for temperature, humidity, light intensity, and soil moisture. The system sends data to ThingSpeak, enabling real-time remote monitoring and analysis, helping users make informed decisions to maintain optimal plant health.

Smart Toilet Flushing

Conventional toilets often rely on manual flushing, which can lead to issues such as improper hygiene, water wastage, and user inconvenience. Additionally, public or shared toilets lack mechanisms to track individual usage, making it difficult to analyze or manage toilet use patterns effectively.
To address these challenges, a smart flushing toilet system was developed using an ESP32 microcontroller and an ultrasonic sensor. The system detects when a user finishes using the toilet, automatically flushes, and dispenses air fragrance for improved hygiene and user experience. Usage data per user is sent to ThingSpeak for monitoring and analysis, enabling better resource management and insights into usage trends.

Weather predict using ANN

Accurately predicting environmental variables, such as temperature and humidity, is crucial for applications in renewable energy, agriculture, and climate analysis. Conventional methods often involve complex statistical models or require extensive domain expertise, which can be time-consuming and less adaptable to dynamic changes in data.
The goal of this project is to develop an Artificial Neural Network (ANN) model using PyTorch to predict temperature and humidity based on time-series input data. The system leverages modern machine learning frameworks to build a robust, adaptive predictive model capable of learning non-linear relationships in environmental data.
